# Workflow Form Builder

A React-based frontend prototype for a Workflow and Dynamic Form Builder application.

## Project Overview

This project allows users to manage dynamic forms and workflow stages through a responsive dashboard interface. The application includes routing, a sidebar layout, dashboard analytics, workflow status tracking, and a chart using static dummy data.

## Tech Stack

- React
- Vite
- Tailwind CSS
- React Router
- Recharts

## Features Completed

- React project setup using Vite
- Tailwind CSS configuration
- React Router setup
- Responsive sidebar and header
- Dashboard homepage
- KPI cards
- Recent forms section
- Workflow status section
- Recharts chart integration
- Mobile responsive layout
